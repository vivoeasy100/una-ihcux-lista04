# una-ihcux-lista04# SistemaRobusto - Prevenção de Erros (IHC)

Quando um programa trava ou fecha sozinho por causa de um erro de digitação, o usuário sente frustração, insegurança e até desconfiança no sistema. Isso pode passar a impressão de que o software é ruim ou mal desenvolvido.

Já quando o sistema trata o erro de forma amigável, explicando o que aconteceu e orientando como corrigir, o usuário se sente mais seguro e confiante. Isso melhora a experiência e evita abandono do sistema.
